One-time setup: install the LeLinc Cookie Grant extension

This lets you log into Instagram, LinkedIn, Facebook and TikTok exactly like you always do — your password never touches LeLinc. Only the resulting login session gets handed over.

Download extension (.zip)
  1. Unzip the download.
  2. Open chrome://extensions in a new tab.
  3. Turn on Developer mode (top right).
  4. Click Load unpacked and select the unzipped folder.
  5. Come back to this dashboard and click Connect again.
